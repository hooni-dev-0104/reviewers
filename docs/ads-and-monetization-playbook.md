# 리뷰콕 광고·수익화 준비 문서

## 1. 지금 시점의 결론

리뷰콕은 아직 초기 제품 단계라서, 수익화는 아래 순서로 가는 게 가장 현실적이다.

1. **직접 제휴/스폰서 슬롯 판매 준비**
2. **디스플레이 광고 네트워크 신청 준비**
3. **제휴 링크형 수익화 추가**
4. 트래픽이 안정화되면 광고 단가/배치 최적화

핵심은 **무조건 빨리 광고를 다는 것보다, 광고 심사를 통과할 수 있는 상태를 먼저 만드는 것**이다.

---

## 2. 추천 수익화 순서

### A. 1순위: 직접 제휴 / 스폰서 슬롯
가장 먼저 붙이기 좋은 방식이다.

이유:
- 현재 서비스 성격상 체험단/마케팅 업주와 연결되기 쉽다.
- 트래픽이 아주 크지 않아도 제안 가능하다.
- 광고 심사 승인에 덜 의존한다.
- UI에 자연스럽게 녹일 수 있다.

현재 리뷰콕에 이미 있는 기반:
- sponsor slot 구조
- source/지역/플랫폼별 타겟팅 여지

### 추천 판매 방식
- 메인 목록 중간 노출 스폰서 카드
- 특정 지역 키워드 검색 결과 상단 노출
- 상세 페이지 하단 추천 스폰서
- 월 단위 고정비 또는 클릭/문의 기반 과금

### 지금 준비해야 할 것
- 광고 상품 소개서 1장
- 광고 위치별 예시 스크린샷
- 예상 월 방문자 / 검색량 / 노출 구조 정리
- 문의용 이메일 / 오픈채팅 / 폼 연결

---

### B. 2순위: 카카오 AdFit
한국 웹서비스에는 현실적인 선택지다.

공식상 확인되는 흐름:
- 매체 등록
- 광고 단위 등록
- 매체 심사
- 광고 게재
- 적립금 정산

장점:
- 국내 친화적
- 운영 가이드가 비교적 명확함
- 웹사이트용 배너/네이티브 운영 가능

주의:
- 콘텐츠 품질, 트래픽 품질, 매체 소유관계가 중요함
- 개인정보처리방침/운영 정책 정리가 필요함
- 심사 통과 전에는 광고 배치와 사이트 완성도가 부족하면 반려 가능

### 리뷰콕에서 AdFit 붙이기 전 체크리스트
- [ ] 개인정보처리방침 페이지
- [ ] 이용약관 또는 서비스 안내 페이지
- [ ] 문의 페이지 / 운영자 연락 수단
- [ ] 모바일/데스크톱에서 광고 자리 명확화
- [ ] 빈 페이지 / 미완성 페이지 최소화
- [ ] 크롤링/집계성 서비스 설명 명확화

---

### C. 3순위: Google AdSense
장기적으로는 붙일 가치가 있지만, 초반에는 승인 난이도가 더 높게 느껴질 수 있다.

장점:
- 글로벌 광고 수요
- 장기적으로 운영 자동화가 쉬움
- 네이티브/자동 광고 실험 여지

주의:
- 승인 전 사이트 완성도 중요
- 얇은 페이지, 중복성 높은 페이지, 가치 설명 부족 시 불리할 수 있음
- 정책 페이지와 소유자 정보가 중요함

### AdSense 신청 전 추천 상태
- 20~30개 이상의 충분한 실제 데이터 페이지가 안정적으로 노출
- 검색/필터 UX 안정화
- 모바일 최적화 완료
- 소개/정책/문의 페이지 완료
- 과도한 placeholder / 빈 링크 제거

---

### D. 4순위: 제휴 링크형 수익화
현재 리뷰콕은 직접 상품 판매형 서비스는 아니지만, 제휴 링크형 수익화는 일부 영역에서 가능하다.

예:
- 체험단 운영 툴
- 콘텐츠 제작 툴
- 블로그/인스타/유튜브 장비
- 예약/촬영/편집 툴

예상 후보:
- 쿠팡 파트너스
- 기타 SaaS 추천 제휴

이 방식은 광고보다 덜 거슬리고, 콘텐츠/가이드 페이지가 많아질수록 잘 맞는다.

## 3. 지금 리뷰콕에 필요한 최소 준비물

### 필수 정책/신뢰 페이지
광고 심사나 제휴 대응 전에 아래는 거의 필수다.

- [ ] 개인정보처리방침
- [ ] 서비스 소개
- [ ] 문의하기
- [ ] 광고/제휴 문의 페이지
- [ ] 데이터 수집/표시 기준 간단 안내

### 광고 상품 정리
직접 영업용으로 최소 이 3개는 만들어야 한다.

1. **광고 상품표**
   - 메인 중간 카드형
   - 검색 결과 상단형
   - 상세 하단 추천형

2. **광고 단가 기준안**
   - 월 고정 노출
   - CPC/문의형
   - 지역 타겟 추가 옵션

3. **매체 소개서**
   - 리뷰콕이 무엇인지
   - 어떤 사용자가 오는지
   - 어떤 순간에 광고가 노출되는지

## 4. 리뷰콕 기준 추천 수익화 로드맵

### 단계 1: 지금 바로
- 직접 제휴 문의 받을 수 있는 구조 만들기
- 스폰서 슬롯을 실제 운영 가능한 상태로 마감
- 정책/문의/소개 페이지 추가

### 단계 2: 트래픽이 조금 더 붙으면
- AdFit 신청
- 광고 위치 1~2개만 제한적으로 테스트
- CTR/이탈률 확인

### 단계 3: 페이지 품질이 더 안정화되면
- AdSense 신청
- SEO성 페이지 / 가이드 페이지 확장
- OG/공유/검색 노출 강화

### 단계 4: 장기
- 지역별 광고 상품화
- 체험단 업주 대상 직접 판매
- 제휴 링크 수익화 결합

---

## 5. 지금 당장 추천하는 선택

리뷰콕의 현재 단계에서는:

### 가장 추천
**직접 스폰서 + AdFit 준비 병행**

이유:
- 당장 수익화 가능성은 직접 스폰서 쪽이 더 높음
- AdFit은 국내 웹매체에 현실적
- AdSense는 조금 더 뒤로 미루는 게 안정적

즉 우선순위는:

1. **광고/제휴 문의 페이지 만들기**
2. **스폰서 슬롯 실운영 가능하게 만들기**
3. **정책 페이지 보강**
4. **AdFit 신청**
5. **추후 AdSense**
